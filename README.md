# The Internet Never Forgets

A professional, visually stunning React + Tailwind CSS web application exploring digital permanence and online privacy.

**CMP 103 Project - GROUP I**  
**Bingham University**

## Features

- **Interactive Hero Section** with animated binary rain and floating elements
- **Data Storage Timeline** showing how information flows through the internet
- **Data Retrieval Simulation** demonstrating how old content resurfaces
- **Google Yourself Tool** with mock search results
- **Educational Case Studies** highlighting real-world consequences
- **Safety Tips** with interactive cards and hover effects
- **Responsive Design** optimized for all devices
- **Cyberpunk Theme** with glowing elements and smooth animations

## Tech Stack

- React 18
- Tailwind CSS
- Framer Motion
- Lucide React Icons
- React Intersection Observer
